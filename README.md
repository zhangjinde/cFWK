c语言framework
本代码库封装了常用的容器、字符串、线程池、异步队列、简单通信协议等常见功能，

Useage:
cd build/
CROSS_FILE=arm-hisiv500-linux.cmake cmake ..
make
make install


备注: 链表  map 字符串 迭代器  io流  日志 单元测试框架  线程池  websocket  隐式模块注册  工厂模型  类似gstreamer那样的管道框架   简便的通信模型   socket的平台无关简化封装 json  ini
